## ADDED Requirements

### Requirement: AuditLogRepository is insert-only

The system SHALL provide `AuditLogRepository` that exposes only `insert()` and `list()` methods. No `update()`, `delete()`, or `soft_delete()` method SHALL exist. The repository SHALL scope all queries to its `tenant_id`.

#### Scenario: Inserting an audit event persists a row
- **WHEN** `AuditLogRepository.insert(actor_id=..., accion="IMPERSONACION_INICIAR", filas_afectadas=1, ip="10.0.0.1", user_agent="Mozilla/5.0")` is called
- **THEN** a new row SHALL exist in `audit_log` with the provided values
- **AND** `fecha_hora` SHALL be set by the database server (not the application)
- **AND** `id` SHALL be a non-null UUID generated by the database

#### Scenario: list() returns rows scoped to the repository's tenant_id
- **WHEN** two different tenants each have audit rows, and a `AuditLogRepository(session, tenant_A_id).list()` is called
- **THEN** only rows belonging to tenant_A SHALL be returned
- **AND** rows from tenant_B SHALL NOT appear in the results

#### Scenario: AuditLogRepository has no update or delete methods
- **WHEN** the `AuditLogRepository` class is inspected
- **THEN** no attribute named `update`, `delete`, `soft_delete`, or `hard_delete` SHALL exist

### Requirement: AuditService.log() records a business event

The system SHALL provide `AuditService.log()` that inserts exactly one row into `audit_log`. It SHALL derive `actor_id`, `tenant_id`, and `impersonado_id` from the `CurrentUser` context, not from separate parameters.

#### Scenario: log() creates an audit entry with correct attribution
- **WHEN** `audit_service.log(current_user=admin_ctx, accion=IMPERSONACION_INICIAR, detalle={"target_user_id": str(target_id)}, filas_afectadas=1, ip="1.2.3.4", user_agent="curl/7.0")` is called
- **THEN** exactly one row SHALL be inserted into `audit_log`
- **AND** `actor_id == admin_ctx.user_id` SHALL be True
- **AND** `tenant_id == admin_ctx.tenant_id` SHALL be True
- **AND** `impersonado_id == admin_ctx.impersonado_id` SHALL be True (if set on the context)
- **AND** `accion == "IMPERSONACION_INICIAR"` SHALL be True
- **AND** `ip == "1.2.3.4"` and `user_agent == "curl/7.0"` SHALL be True

#### Scenario: log() rejects an unknown action code (RN-24)
- **WHEN** `audit_service.log(..., accion="CODIGO_ARBITRARIO_INVALIDO")` is called
- **THEN** `ValueError` SHALL be raised with a message identifying the invalid code
- **AND** no row SHALL be inserted into `audit_log`

---

### Requirement: AuditService.list() returns paginated, filtered results

The system SHALL provide `AuditService.list()` that returns a `AuditLogListResponse` with pagination metadata. It SHALL enforce scope-based filtering before applying any optional user-supplied filters.

#### Scenario: list() with scope='all' returns all entries for the tenant
- **WHEN** `audit_service.list(tenant_id=t, scope="all", current_user_id=admin_id, page=1, page_size=50)` is called
- **AND** there are 5 audit rows for the tenant
- **THEN** all 5 entries SHALL appear in `items`
- **AND** `total == 5` SHALL be True

#### Scenario: list() with scope='own' restricts results to current_user_id's actions
- **WHEN** `audit_service.list(tenant_id=t, scope="own", current_user_id=coord_id, page=1, page_size=50)` is called
- **AND** the tenant has 3 rows from coord_id and 2 rows from another actor
- **THEN** only the 3 rows where `actor_id == coord_id` SHALL be returned
- **AND** `total == 3` SHALL be True

#### Scenario: list() with accion_filter returns only matching entries
- **WHEN** `audit_service.list(tenant_id=t, scope="all", current_user_id=admin_id, accion_filter="IMPERSONACION_INICIAR")` is called
- **AND** the tenant has rows with accion="IMPERSONACION_INICIAR" and "IMPERSONACION_FINALIZAR"
- **THEN** only rows with `accion == "IMPERSONACION_INICIAR"` SHALL be returned

#### Scenario: list() paginates correctly
- **WHEN** there are 15 rows and `list(..., page=2, page_size=10)` is called
- **THEN** `len(items) == 5` SHALL be True
- **AND** `total == 15`, `page == 2`, `page_size == 10` SHALL be True
